# Global Dream

A simple web application with frontend and backend.

## About

This is a Quiz Challenge application that tests users on web development basics (HTML, CSS, JavaScript). Users can answer multiple-choice questions, navigate between questions, and see their score at the end.

### Features

- Multiple-choice quiz questions
- Progress tracking with navigation dots
- Instant feedback on answers
- Score calculation and results display
- Keyboard navigation support (Tab and Enter)
- Progress saved locally

### Backend API

The backend provides a REST API for course management:
- Create, read, update, delete courses
- Track user progress
- Quiz scoring system
- API key authentication

## Project Structure

- `frontend/` - HTML, CSS, and JavaScript files
- `backend/` - Node.js server

## Setup

### Backend

1. Navigate to the backend folder:
   ```
   cd backend
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the server:
   ```
   node server.js
   ```

### Frontend

Open `frontend/index.html` in your browser.

## Technologies

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
